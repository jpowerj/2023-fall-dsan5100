What Things Have Distributions?

  • Answer: Random Variables

CDFs/pdfs/pmfs: What Are They?

  • Functions which answer questions about a Random Variable (\(X\) in this case)
  • CDF: What is probability of value less than or equal to \(X\)?

\[ F_X(v_X) \overset{\text{defn}}{=} \Pr(X \leq v_X) \]

  • pmf: What is the probability of this exact value? (Discrete only)

\[ p_X(v_X) = \Pr(X = v_X) \]

  • pdf: …It’s the thing you integrate to get the CDF

\[ f_X(v_X) = \frac{d}{dv_X}F_X(v_X) \iff \int f_X(v_X)dv_X = F_X(v_X) \]

CDFs/pdfs/pmfs: Why Do We Use Them?

  • CDF is like the “API” that allows you to access all of the information about the distribution (pdf/pmf is derived from the CDF)
  • Example: we know there’s some “thing” called the Exponential Distribution…
  • How do we use this distribution to understand a random variable \(X \sim \text{Exp}\)?
    • Answer: the CDF of \(X\)!
    • Since all exponentially-distributed RVs have the same PDF, we can call this PDF “the” exponential distribution
  • Say we want to find the median of \(X\): The median is the number \(m\) satisfying

\[ \Pr(X \leq m) = \frac{1}{2}, \; \Pr(X \geq m) = \frac{1}{2} \]

Finding a Median via the CDF

Median of a Random Variable \(X\)

The median of a random variable \(X\) with some CDF \(F_X(v_X)\) is the [set of] numbers \(m\) for which the probability that \(X\) is lower than \(m\) is the same as the probability that \(X\) is greater than \(m\):

\[ \begin{align*} \text{Median}(X) &= \left\{m \mid F_X(m) = 1 - F_X(m) \right\} \\ &= \left\{m \mid \int_{-\infty}^{m}f_X(v_X)dv_X = \int_{m}^{\infty}f_X(v_X)dv_X \right\} \end{align*} \]
